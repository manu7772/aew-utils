# Aequation Wire Bundle

> Ce bundle installe une boîte à outils de asset/controllers pour des fonctionnalités **Symfony 7.2**.
> -
> Par Emmanuel Dujardin / Aequation Webdesign

## Install

*In terminal, type:*

$ symfony composer require aequation/utils
